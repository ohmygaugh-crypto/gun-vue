useAccount(pub) ⇒ Account #

Load and handle user's account by a public key

ParamTypeDescription
pubrefThe public key of a user as a string or a ref

Example

import { useAccount } from '@gun-vue/composables'

const pub = 'XnpLVDYZWdl1NNgo6BlD6e3-n3Fzi-ZzVrzbIgYCYHo.9-hHUHaWNaAE6tMp800MMzNtDLtjicS53915IrBu4uc'

const { account } = useAccount(pub)

encFor(data, sender, receiver) ⇒ String #

Encrypt data for one receiver entity

  1. Generates encryption secret using bob's epub and current user pair
  2. Enctypts data with this secret

Returns: String - Encrypted data string to be sent

ParamTypeDescription
dataStringStringified data to be encrypted
senderEntityAn object with pub and epub strings - the user.is object of the reciever's account
receiverSEAPairSEA Pair of the sender – epriv key will be used to encrypt the data

decFrom(data, sender, receiver) ⇒ String #

Decrypt a private message from an entity

  1. Generates secret using senders epub and current user pair
  2. Decrypts the data with this secret

Returns: String - Decrypted data

ParamTypeDescription
dataStringEncrypted private data
senderEntityAn object with pub and epub strings - the user.is object of the sender's account
receiverSEAPairSEA Pair of the receiver – epriv key will be used to encrypt the data

Entity : Object #

Properties

NameTypeDescription
pubStringthe public key
epubStringthe elliplic encryption epub

useZip() ⇒ useZip #

Zip file creation toolbox

Example

import {useZip} from '@gun-vue/composables'
const { zip, zipPost, addMd, addFile, downloadZip } = useZip()

addFile(options) ⇒ String #

Add a binary file to the zip

Kind: inner method of useZip
Returns: String - the resulting filename

ParamType
optionsObject

Example

if (post.cover) { // a base64 encoded picture
  const fileName = await addFile({
    title: "cover",
    file: post.cover,
    folder: post.title,
  });
  post.cover = fileName;
}
