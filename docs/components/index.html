Components #

Pluggable Vue 3 + Gun DB UI components

How to install #

  1. Install the components library:
npm i @gun-vue/components
  1. Import an async component into your Vue 3 SFC script setup:
import { defineAsyncComponent } from 'vue'

const RelayPulse = defineAsyncComponent(() =>
  import('@gun-vue/components')
)
  1. Use the component in your SFC template:
<template>
<relay-pulse />
</template>
  1. Have fun combining different components with a shared Gun state.
